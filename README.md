# Angular – Estudos Fundamentais

## Sobre
Repositório criado exclusivamente para estudo prático do framework Angular,
explorando seus principais conceitos, arquitetura e padrões.

## Objetivo
- Aprender Angular do zero
- Entender a estrutura de um projeto Angular
- Compreender como o framework organiza componentes, services e módulos
- Comparar o modelo do Angular com React/Next.js

## Conteúdos Abordados
- Componentes e Templates
- Data Binding (One-way e Two-way)
- Diretivas
- Services e Dependency Injection
- Lifecycle Hooks
- Rotas e Lazy Loading
- Consumo de APIs com HttpClient
- Forms (Template-driven e Reactive Forms)

## Estrutura do Projeto
- Separação de responsabilidades
- Organização por módulos
- Services para regras de negócio
- Componentes reutilizáveis

## Status
Em desenvolvimento contínuo (estudo)

## Observação
Este repositório não representa um produto final ou sistema de cliente.
Trata-se de um laboratório de aprendizado focado em compreender o funcionamento
do Angular em profundidade.
